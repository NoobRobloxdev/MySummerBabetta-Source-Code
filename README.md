🛵 My Summer Babetta (ALPHA)
A rural Slovak simulator built with C++ and Raylib. Build, maintain, and ride your legendary Babetta through a village while managing your survival and money.

[!CAUTION]
COPYRIGHT NOTICE: This project is built by LOM_Noob and Jurmat. Do not republish this code on GitHub or any other platform.

🛠 Features
Modular Assembly System: The Babetta is not just a vehicle; it's a project. It will not move unless you have manually installed:

The Engine ⚙️

Front Tire 🛞

Rear Tire 🛞

Detailed World: The Garage: 4-wall structure with shelves for part storage.

Solid Collisions: Buildings are physical objects. You can walk into the garage, but you can't walk through walls.

Survival Mechanics: Real-time Hunger and Thirst tracking. (WIP)

Economic System: Start with 2000 EUR.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎮 Controls
Key	Action
W / A / S / D	Walk / Drive Babetta
Mouse	Look around
Left Click	Pick up part / Install part / Pull part off bike
E	Mount or Dismount Babetta
ESC	Toggle Pause Menu / Show Cursor
F6	Toggle FPS Counter
F9	Cheat: Instant 999,999 EUR

🏗 Technical Details
Language: C++

Library: Raylib 5.0

Compiler: MinGW-w64 (GCC)

Physics: Custom ray-cast camera logic and state-based vehicle movement.

🛑 Troubleshooting "Permission Denied"
If you receive a ld.exe: cannot open output file build/Game.exe: Permission denied error during compilation, it means the game is still running.

Open Task Manager.

Find Game.exe.
If not then create a report a bug or just contact this email: vondranoob233@gmail.com

Click End Task and try compiling again.
